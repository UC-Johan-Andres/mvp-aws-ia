Empresas

El registro aplica a n8n (asignación en almacén local) y a LibreChat (campo company en MongoDB). Puedes guardar API keys por proveedor (OpenAI, Gemini); al crear o actualizar la empresa se dispara la sync en segundo plano hacia LibreChat (keys) y n8n (credenciales en el proyecto personal de cada usuario de esa empresa). Renombrar empresa también re-sincroniza. Manual: POST /gestion/api/companies/sync con {"name":"empresa"}. En n8n, la credencial creada por el launcher se llama como Launcher AI · OpenAI (ChatGPT) · tu@email; al abrirla para editar, el campo API Key suele verse vacío (n8n no vuelve a mostrar secretos guardados): es normal. Comprueba que exista en la lista de credenciales y ejecuta un nodo OpenAI con esa credencial. La primera carga puede tomar valores de GESTION_COMPANIES si el JSON aún no tenía lista.

Opcional — claves solo en este navegador; no se muestran completas después de guardar.

Nombre: letras, números, espacio, -, _ y . (máx. 64).
